# Example plugin

This package holds a reference **`tabocalypse-plugin.json`** you can import in Tabocalypse (Settings → Import declarative plugin).

- Types and validation live in **`@tabocalypse/plugin-sdk`** (`packages/plugin-sdk`).
- To author a new plugin, copy this folder or the JSON file, edit IDs/widgets, validate with the extension UI or a small script using `validatePluginJsonText` from the SDK.
